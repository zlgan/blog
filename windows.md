# 电源管理

```powershell
powercfg /hibernate on #启用休眠
powercfg /hibernate off #禁用休眠
```





# 账户管理

```powershell
#启用管理员账户
net user Administrator /active:yes
net user Administrator /active:yes


#启用/禁用密码要求
net user Administrator /passwordreq:yes

```
