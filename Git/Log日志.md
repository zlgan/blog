### git log
```bash
     git log  # 查看log日志 
     git log --pretty=oneline #显示完整的hash
     git log --oneline #显示hash的前七位
     git log --author=zlgan #过滤作者
     git log -3 #查询最近三次提交
     git log  --grep=keywords #过滤comment包含特定关键字的提交记录 
```
