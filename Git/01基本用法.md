### 查看提交的内容

```bash
git show []
```
- 常用的命令

  ```bash
  #追加提交
  git commit --amend
  ```
- ###### 代码比较

  ```bash
  #工作区与暂存区比较
  git diff 
  #工作区与版本库比较
  git diff head 
  #比较State与版本库
  git diff --cached
  ```

### 查看提交记录
```bash
    #查看最后一次提交的修改记录
    $ git show
    #最后一次修改的文件列表  
    $ git show --name-only
```


### 注意事项
```bash
  git commit --amend -m '修改后的注释' #追加提交，可以用来修改注释
  git mv #文件重命名，可以只修改大小写
  git rm #删除文件
```
