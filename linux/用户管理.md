### 查找指定用户是否存在

```BASH
[root@node11 node]# grep node -n /etc/passwd
47:node:x:1000:1000:node1:/home/node:/bin/bash
```

### 切换用户

```BASH
#root用户切换到普通用户
[root@node11 node]# su node
#普通用户切换到root用户
[node@node11 ~]$ su
密码：
[root@node11 node]# 
```
