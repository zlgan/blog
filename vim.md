# normal

```
kjhl 上下左右
gg 行首
G 行尾
dd 删除当前行
10dd 删除第10行

ctrl+u 向上翻页
ctrl+d 向下翻页


:set nu  设置行号
```



```
i "insert" 在光标开始之前输入
I 在本行的开头开始输入
a "append"
o 下方插入
O 上方插入

s删除当前光标字符，然后开始输入
S删除当前行，然后重新输入

```
